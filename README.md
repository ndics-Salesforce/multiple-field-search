# コンポーネントの説明

- <b>複数項目(取引先名・業種・種別)から取引先を検索するコンポーネントです。</b><br>
- 3つの項目のうち、どれか1項目でも当てはまると検索結果に表示されます。（OR検索）<br>
- ある項目を空欄にして検索すると、その項目は検索対象にはなりません。
